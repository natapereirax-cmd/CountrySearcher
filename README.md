# Country Searcher

Esse projeto consiste em um site que consome uma API externa capaz de fornecer dados acerca de determinado país. Uma vez que o usuário digita o nome de um país, se este de fato existir e estiver dentro do banco de dados da API externa, os dados serão apresentados em uma caixa no lado direito da tela do site.

---
## Conhecimentos aplicados

* Estrutura de dados Python
* Criação de aplicações web com Flask
* Definição de páginas/endpoints da aplicação
* Manipulação de formulários HTML
* Consumo de APIs REST
* Tratamento das respostas HTTP
* Tratamento de exceções
* Manipulação de JSON
